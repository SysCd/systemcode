# System Code

**Live Site:** [https://systemcode.net](https://systemcode.net)

Practical software and infrastructure, built from first principles.

System Code is the public website for independent software, infrastructure tools, AI-focused products, developer utilities, and systems documentation.

System Code Software is the business/trading name behind the public System Code brand. The site is structured so the business name can be updated after company registration.

## Site Focus

- Infrastructure tools
- AI products
- Developer utilities
- Systems documentation
- Portfolio context for Microsoft infrastructure, cloud, automation, and AI systems work

## Deployment

- **Output directory:** `build`
